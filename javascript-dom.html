What is the DOM

The DOM is a tree-like representation of the web page that gets loaded into the browser. It represents the web page using a‌‌ series of objects.

By using the DOM we can access and change elements by specifiying an ID, class or element name. One way you can interact with the DOM is by using JavaScript is to change an HTML element. The following code selects the element class="test"

document.getElementByclass('test')

You can grab the element with the class of 'test' and change its inner HTML text.

document.getElementByclass('test').innterHTML = "new text"
